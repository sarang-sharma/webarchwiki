=WebarchWiki
Wiki is an collaborative database for tutorials and articles on topics such as App Development, Web Development, Web Designing, Graphic Designing, VFX etc.

== Features
* CRUD
* Tags (acts-as-taggable-on)
* Markdown (Redcarpet)
* Syntax Highlighting (Pygments)
* User Authentication (via [Devise](https://github.com/plataformatec/devise))
